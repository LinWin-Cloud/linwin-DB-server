# Python Api

### 安装依赖
```commandline
pip install -r requirements.txt
```

### 示例
```python
from drive import driver
db = driver.Driver(
    host="127.0.0.1",
    username="root",
    password="123456",
)

try:
    db.createDatabase("Linwin")
    db.createData("Data", "Linwin")
except driver.DatabaseError as e:
    ...
```
